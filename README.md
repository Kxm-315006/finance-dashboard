# 💰 Finance Dashboard

A clean and interactive frontend dashboard for tracking financial activity, built using React and Tailwind CSS.

## 📌 Features

### 📊 Dashboard Overview

* Total Balance, Income, Expenses summary cards
* Responsive UI with clean layout

### 💳 Transactions

* View all transactions
* Search by category
* Filter by:

  * Category
  * Type (Income/Expense)
  * Date range
* Limit number of results
* Export transactions as CSV

### 👤 Role-Based UI

* **Viewer** → Can only view data
* **Admin** → Can add new transactions

### 💡 Insights

* Highest spending category
* Monthly comparison
* Overall financial status

### 🌙 Dark Mode

* Toggle between light and dark theme

### 💾 Data Persistence

* Transactions saved using Local Storage

## ⚙️ Setup Instructions

### 1. Clone the repository

```powershell
git clone https://github.com/YOUR_USERNAME/finance-dashboard.git
cd finance-dashboard
```

### 2. Install dependencies

```powershell
npm install
```

### 3. Run the project

```powershell
npm run dev
```
